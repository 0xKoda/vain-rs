# Vanity Address Generator 

An ethereum vanity address generator built with rust. 

## Usage

Clone the repository
```
git clone https://github.com/0xKoda/vain-rs.git
cd vain-rs
```

Build the crate
```
Cargo Build
```

Run the program
```
Cargo run -p <your-desired-pattern>
```

Example
```
cargo run -p 1234
```
